# Portfolio (Hugo)  

Este repositório contém meu **segundo portfolio** que foi desenvolvido usando o **Hugo**, um gerador de sites estáticos em Go 
O projeto foi feito como aprendizado e prática em desenvolvimento web, criação de sites estáticos e organização de projetos pessoais


## Sobre o Portfolio
- Site construído com **Hugo** e tema **Spectral**  
- Conteúdo em **português** com algumas seções multilíngues  
- Demonstração de projetos, habilidades e experiências profissionais  

### Projetos Destacados
- **FipeAPI:** API REST em Java para consulta de preços de veículos com base na Tabela Fipe ([repositório](https://github.com/Gerfy1/FipeAPP))  
- **MedAPI:** API RESTful para gerenciamento de consultório médico em Java ([repositório](https://github.com/Gerfy1/MedAPI))  
- **Forum HUB:** Backend de fórum com Java, Spring e MySQL ([repositório](https://github.com/Gerfy1/ForumHub)) 

## Tecnologias Utilizadas
- Hugo (gerador de sites estáticos)  
- HTML, CSS, JavaScript  
- Markdown para conteúdo  
- Tema: Spectral
  
## Observações
Este repositório serve como histórico do meu aprendizado em criação de portfólios e desenvolvimento web estático  
Não é mais utilizado como meu portfolio principal
